Inside your project, create folder:

.github/workflows/


Create file:

ci.yml

Example Basic CI File:
name: Simple CI

on:
  push:
    branches: [ main ]

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - name: Checkout code
        uses: actions/checkout@v3

      - name: Print message
        run: echo "CI is running successfully!"


Now whenever you push to main, GitHub runs this automatically.
